Obtener horas disponibles de un Terapeuta

Endpoint:

Recibe, en formato JSON:

  • selectedDay: string
  • psy: string, id
  • duration: string

Body Ejemplo:

{
  "selectedDay": "19/10/2020",
  "psy": "5f8d3dd25418f6bd7741a03e",
  "duration": 50
}

Respuesta Ejemplo:

{
    "hours": [
        "12:00",
        "14:00",
        "17:00",
        "20:00"
    ],
    "message": "Schedule obtained correctly"
}
